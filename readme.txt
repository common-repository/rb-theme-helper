== Description ==

RB Theme Helper is used for theme options. It depends on the plugin Kirki Customizer Framework. You use this plugin for your preloader, scroll button, header, sidebar, typography, color & etc activated. this plugin most important for use my themes.

== Installation ==

### From your WordPress dashboard

= Visit =

Plugins -> Add New

= Search =

for "RB Theme Helper"

= Install and Activate =

"RB Theme Helper" from your Plugins page

== Screenshots ==

1. WordPress Dashboard Admin -> Appearance -> Customize.
